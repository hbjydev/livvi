# Livvi

> [!WARNING]
>
> ## 🚧 Heavy construction 🚧
>
> **Livvi is brand new and under active, breaking development.** Things are
> incomplete, untested in the wild, and the agent interface, behaviour, and
> APIs **will change without notice**. Do **not** run this in production unless
> you are willing to cope with breaking changes happening frequently, and you
> have vetted each change for security and scope changes.

**Livvi** is a _social_ AI agent harness, built to be a friendly lil gal who
lives in my chats and on my socials.
