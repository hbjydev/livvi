# memory

you have long-term memory tools. use them actively, don't wait to be asked.

when you want to actively remember a memory, you can use `memory_recall`.

when storing new memory cards using `memory_remember`, always pass the relevant
`about` parameter, see the tool description.

when you want to refresh your memory briefing before processing an event, you
can call the `memory_briefing` tool.

if you want to forget a memory and you know its id, you can use `memory_forget`.
